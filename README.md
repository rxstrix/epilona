# Epilona — Сайт салона лазерной эпиляции

Одностраничный сайт для салона лазерной эпиляции. Подарок на 8 марта 💜

## Запуск

```bash
npm install
npm run dev
```

Откройте http://localhost:5173

## Сборка

```bash
npm run build
```

Готовые файлы будут в папке `dist/`

## Настройка

- **Название салона** — замените "Epilona" в App.tsx на нужное
- **Телефон** — обновите номер в секции контактов
- **Цены** — скорректируйте в секции "Цены"
- **Соцсети** — добавьте ссылки на WhatsApp, Telegram
